# AI-Based Agricultural Engineering Calculator

This project is a web-based application designed to perform agricultural engineering calculations using basic AI-driven logic. It helps simplify farm-related computations and improves efficiency.

## Features
- Automated agricultural calculations  
- User-friendly interface  
- Fast and accurate results  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Purpose
To demonstrate how AI can be applied in agriculture for smarter decision-making.
